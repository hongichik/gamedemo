# Animal Listening Game (Demo)

Simple HTML/CSS/JS demo: listen to an English word and pick the matching picture/word.

How to use:

1. Open `index.html` in your browser.
2. Click the speaker to hear the target word (auto-plays at each round).
3. Click the correct image/word.

Rules:

- Correct: +10 points and a 👍 indicator.
- Wrong: -5 points and lose one heart (3 hearts total).
- Win: finish all words.
- Lose: hearts reach 0.

Notes:

- The demo tries to fetch pronunciation audio from the Dictionary API and falls back to your browser's SpeechSynthesis if not available.
- Images are loaded from Unsplash by query (random results).
